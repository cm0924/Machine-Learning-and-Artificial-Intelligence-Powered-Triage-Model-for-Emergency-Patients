🏥 AI-Powered Emergency Triage Prediction Model
Specialism: Artificial Intelligence | APU Final Year Project 

📌 Project Overview
Emergency Department (ED) overcrowding in Malaysian public hospitals is a critical crisis that leads to increased mortality. This project provides a Techno-Functional solution: a Machine Learning-driven Triage Model that predicts patient severity (ESI) upon arrival. By automating the initial assessment, the system ensures high-risk patients receive immediate care while optimizing limited hospital resources.

🚀 Key Features
Intelligent Severity Prediction: Leverages a Random Forest classifier to categorize patients into five Emergency Severity Index (ESI) levels.

Data-Driven Vital Analysis: Trained on a preprocessed clinical dataset including Heart Rate, SpO2, and Blood Pressure.

GenAI Medical Reasoning: Includes an AI Reasoning engine that explains the "Why" behind a prediction to assist medical staff in decision-making.

Interactive Web Dashboard: A functional tool for medical staff to input data, view real-time triage recommendations, and analyze patient data through visualization.

High-Performance Accuracy: Achieved an 86% accuracy rate in predicting resource needs, significantly outperforming traditional manual triage methods.

🛠 Tech Stack
Language: Python.

Machine Learning: Scikit-learn (Random Forest, SVM), Pandas, NumPy.

Frontend & Analytics: Matplotlib for data visualization.

Deployment: Flask / Streamlit for the web-based diagnostic tool.

AI-Powered Emergency Triage Prediction Model
Specialism: Artificial Intelligence | APU Final Year Project

📌 Project Overview
This project addresses the critical issue of overcrowding in Malaysian public hospital Emergency Departments (ED). I developed a Machine Learning-driven Triage Model that predicts patient severity levels (ESI) upon arrival, ensuring that high-risk patients receive immediate care while optimizing hospital resource allocation.

🚀 Key Features
Severity Prediction: Uses a Random Forest classifier to categorize patients into five Emergency Severity Index (ESI) levels.

Data-Driven Insights: Trained on a preprocessed dataset of vital signs (Heart Rate, SpO2, Systolic/Diastolic BP) and demographic data.

Web Interface: A functional tool for medical staff to input patient data and receive an instant triage priority recommendation.

High Accuracy: Achieved an 86% accuracy rate in predicting "High Resource" vs. "Low Resource" patients, significantly outperforming traditional manual triage.

🛠 Tech Stack
Language: Python.

Libraries: Scikit-learn (Random Forest, SVM), Pandas, NumPy, Matplotlib.

Deployment: Flask/Streamlit for the web-based diagnostic tool.
